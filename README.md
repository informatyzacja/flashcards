## Getting started

### Requirements

#### Development

- Volta

### Installation

```bash
git clone https://github.com/informatyzacja/flashcards.git
cd flashcards
pnpm install --frozen-lockfile
```

## Usage

Run in development mode:

```bash
pnpm dev
```

or start the app:

```bash
pnpm start
```
